# Aurelia Suites - Hotel Management System

## Requirements
- XAMPP (Apache + MySQL)

## Setup
1. Install XAMPP
2. Start Apache and MySQL
3. Open phpMyAdmin → Import → اختاري `database.sql`
4. Copy project folder to `C:\xampp\htdocs\hotel_project`
5. Open browser: `localhost/hotel_project/index.html`

## API Endpoints
- GET  /api/services
- POST /api/reservation_services
- DELETE /api/reservation_services
- POST /api/feedback
- GET  /api/feedback
